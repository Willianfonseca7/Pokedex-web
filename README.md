Pokémon search app built with vanilla HTML, CSS and JavaScript, consuming the [PokéAPI](https://pokeapi.co/). Search by name or ID and see stats, type, height, weight and abilities — no frameworks, no libraries.

---

## ✨ Features

- 🔍 Search Pokémon by name or ID
- 🖼️ Displays image, type, weight, height and abilities
- ⬅️➡️ Navigate between Pokémon (previous / next)
- ⚠️ Error handling for Pokémon not found
- 📱 Responsive layout — works on mobile and desktop

---
